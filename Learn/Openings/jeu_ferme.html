Le Gambit Dame

L'ouverture la plus célèbre du jeu de pions Dame. Un classique indémodable pour dominer le centre.

1.d4d5 2.c4
Stratégie

Le Plan

Les Blancs attaquent le pion central d5 avec un pion latéral (c4). L'idée est d'échanger le pion c contre le pion d pour dominer le centre.

Avantages

  • Lutte immédiate pour le centre
  • Développement naturel

Inconvénients

  • Les Noirs ont des défenses solides
  • Demande de la technique
